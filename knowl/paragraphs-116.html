Troubleshooting.

If your code won’t compile:
  • If you are using overleaf, it will help you find errors in your code
  • Make sure commands use a backslash [\] rather than a forward slash [/]
  • You may be missing a dollar sign
  • Check your spelling on commands [ie. \epslion rather than \epsilon]
  • You may be missing a starting or ending bracket \(\{\textrm{or}\}\)
  • If you have a \begin command, you need an \end command
  • remove spaces beween lines if you are in an itemize/list/align environment
If you don’t know the code for a particular symbol/object/format:
  • If you don’t know a symbol or how to format something - google it. "how to write a matrix in latex" etc.
  • You can copy/paste others code
  • If code you find online wont run - look at their preamble information to see if they used a package you have not included
Ask for help if you can’t figure something out!
