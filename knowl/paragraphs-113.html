Getting Started in Overleaf.

  1. Create an account on https://www.overleaf.com
  2. You can copy/paste or upload the HW template document found on canvas
  3. Save a copy of this for starting a homework assignment so that the original copy remains as a template in your files
  4. Edit the existing template, copy/paste or use the same text to add new sections, proofs, etc.
  5. After you make changes, click "recompile" and the pdf will update and save
  6. A cool thing about overleaf is that you can have different people sharing and working on a document together by sharing a document link
